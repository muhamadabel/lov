## KONTEKS PROJECT

Kamu sedang membangun **LOVie** — sebuah website personal yang dibuat sebagai ungkapan terima kasih yang tulus. Ini bukan website biasa. Ini adalah pengalaman sinematik dan emosional. Setiap scroll harus terasa seperti membalik halaman buku. Setiap animasi harus lambat, disengaja, dan intim.

**Referensi visual utama:** https://clarte.page  
Perhatikan cara mereka menggunakan tipografi serif besar, whitespace ekstrem, dan scroll yang terasa seperti film — bukan website.

---

## TECH STACK

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animasi:** Framer Motion (WAJIB untuk semua scroll animation)
- **Font:** Cormorant Garamond (display/heading) + DM Sans (body) — import dari Google Fonts
- **Hosting:** Vercel

### SECTION 1 — HERO

**Layout:** Full viewport height. Teks di tengah secara vertikal.

**Konten:**
```
LOVIE                          ← font: Cormorant Garamond, weight 700, ~120px, uppercase
God of 35                      ← font: Cormorant Garamond, weight 300 italic, ~64px
```

**Animasi:**
- "LOVIE" muncul fade + slight scale dari bawah, delay 0.3s
- "God of 35" muncul word by word, tiap kata delay 0.15s setelah kata sebelumnya
- Durasi tiap animasi: 1.2s, ease: `[0.16, 1, 0.3, 1]`
- Scroll indicator (chevron kecil atau titik) muncul di bawah setelah 2s, gentle bounce loop

**Background:**
- Solid `#AB1509`
- Tambahkan subtle vignette: radial gradient dari `rgba(0,0,0,0)` di tengah ke `rgba(0,0,0,0.35)` di pinggir

### SECTION 2 — ABOUT HER

**Layout:** Dua kolom. Kiri: image placeholder (foto animasi 1). Kanan: heading + body text.

**Konten kanan:**
```
VIE                            ← Cormorant Garamond, bold, ~80px
Puput Octaviana Siam           ← DM Sans, 12px, letter-spacing lebar, opacity 0.6

[teks about her — lihat content.md]
```

**Animasi:**
- Image kolom kiri: fade in dari kiri, slide 40px → 0, saat section masuk viewport
- "VIE" heading: fade in dari kanan
- Teks body: reveal per baris (line by line), delay 0.1s antar baris
- Semua trigger saat elemen masuk viewport (Framer Motion `whileInView`)
- `viewport={{ once: true, margin: "-100px" }}`

**Catatan foto:**
- Gunakan placeholder `<div>` dengan label "image animasi 1" untuk sekarang
- Dimensi: ~400x500px, background `rgba(255,247,211,0.1)`, border `1px solid rgba(255,247,211,0.2)`

### SECTION 7 — LOGIN GATE (PALING BAWAH)

**Konsep:**  
Section ini **tersembunyi** sampai user benar-benar scroll ke paling bawah — terinspirasi dari cara clarte.page menyembunyikan section terakhirnya. Rasanya seperti menemukan sesuatu yang tidak semua orang akan lihat.

**Reveal mechanic:**
- Section awalnya `opacity: 0` dan `scale: 0.95`
- Baru muncul saat scroll progress mencapai 95%+ dari total halaman
- Atau: gunakan Framer Motion `useScroll` + `useTransform` untuk tie opacity ke scroll position
- Tambahkan blur yang perlahan hilang: `filter: blur(8px) → blur(0px)` seiring scroll

**Layout:**
- Animasi hati kecil melayang — 3–5 hati kecil dengan float animation loop, opacity 0.3–0.6
- Input username dan password, styling minimal:
  - Background: `transparent`
  - Border: `1px solid rgba(255,247,211,0.3)`
  - Text: `#FFF7D3`
  - Placeholder: `rgba(255,247,211,0.4)`
  - Padding: `12px 16px`
  - Border-radius: `4px`

**Button LOGIN:**
> ⚠️ GUNAKAN KODE GLOW BUTTON YANG SUDAH ADA (file: `reference-glow-button.tsx`)  
> Jangan buat button biasa. Terapkan glow button dari referensi tersebut dengan warna disesuaikan ke cream `#FFF7D3`.

**Logic auth:**
```javascript
const CREDENTIALS = {
  username: "AB1509",   // ← username
  password: "35"        // ← password
}
```
- Validasi: client-side saja, tidak ada backend, tidak ada database
- Jika salah: shake animation `x: [-10, 10, -5, 5, 0]`, duration 0.4s + teks error kecil
- Jika benar: `AnimatePresence` → fade out login form → fade in FarewellPage component

## ATURAN ANIMASI (WAJIB DIIKUTI)

1. **Semua animasi LAMBAT** — minimum duration 0.8s, ideal 1.2s
2. **Tidak ada yang tiba-tiba** — selalu ada ease in/out
3. **Ease default:** `[0.16, 1, 0.3, 1]` (custom bezier, sinematik)
4. **Scroll-driven:** gunakan `whileInView` + `viewport={{ once: true }}`
5. **Stagger:** elemen dalam satu group muncul berurutan, delay 0.1–0.15s
6. **Tidak ada bounce** kecuali scroll indicator di hero
